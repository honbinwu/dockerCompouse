# Final presentation

# Start all service:

```sh
$ docker-compose up -d
```

# Stop all service:

```sh
$docker-compose down

```

# Start one service:

```sh
$docker-compose start $SERVICE_NAME 
```

# Stop one service:

```sh
$docker-compose stop $SERVICE_NAME
```

# Access a running container:

```sh
$docker-compose exec $SERVICE_NAME bash
```
